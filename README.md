# README

## Prerequisite

```bash
npm init
npm install typescript --save-dev
export PATH=/tmp/test/node_modules/.bin:$PATH
tsc --init
npm install express --save
npm install ts-node nodemon @types/node @types/express --save-dev
```

## Development

```bash
npm run dev
```

## Compile

```bash
npm run build
```

## Run

```bash
npm start
```
